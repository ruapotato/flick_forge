How apps are licensed and distributed on Flick Store

App Licensing

AGPL-3.0 License

All apps published on Flick Store are released under the GNU Affero General Public License version 3 (AGPL-3.0).

This means:

  • All apps are free and open source
  • You can view, modify, and redistribute the source code
  • Any modifications must also be released under AGPL-3.0
  • The source code is available on the Flick Project GitHub

Flick Project Ownership

All apps are published under the Flick Project.

When you submit a request for an app:

  • The resulting app becomes part of the Flick Project
  • The app is owned by the Flick Project community
  • You retain credit as the original requester
  • Anyone can contribute improvements to any app

AI-Generated Apps

Apps on Flick Store are generated by AI (Claude) with human oversight:

  • Apps are created from user-submitted prompts
  • A human reviews and approves all apps before release
  • Apps are tested in "Wild West" before promotion to stable
  • Community feedback helps improve app quality

Qt/QML Only

All apps are built as Qt/QML applications for the Flick mobile shell:

  • Native Qt/QML for optimal performance
  • Touch-friendly, mobile-first design
  • Consistent look and feel across all apps
  • No GTK, Electron, or web-based apps
